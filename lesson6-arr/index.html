Arrays

Definition

                    let arr = new Array()
                    let arr = []

Index

                    let arr = [`str`, 19, null, undefined, false]
                    arr[0] => `str`
                    arr[1] => 19
                    arr[2] => null
                    arr[3] => undefined
                    arr[4] => false

Objects in array

                    let arr = [() => {alert(`WooW`)}, {login: `login`, pass: 777}]
                    arr[0]()
                    arr[1].login

FOR

                    for (item of array) {...}

                    const timer = [`five`, `four`, `three`, `two`, `one`]
                    for (item of timer) {
                        console.log(item)
                    }

Methods of arrays

splice

                    arr.splice(start, count, …new elements)

                    const arr = [1, 2, 3, 4, 5, 6]
                    const result = arr.splice(1, 2, 23, 32, 24)
                    console.log(arr);      // [1, 23, 32, 24, 4, 5, 6]
                    console.log(result);   // [2, 3]

slice

                    arr.slice(start, end)

                    const arr = [`a`, `b`, `c`, `d`, `e`, `f`]
                    const result = arr.slice(2, 4) // [`c`, `d`]

concat

                    arr.concat(arr1, arr2)

                    const arr = [1, 2]
                    const result = arr.concat([3, 4], [5, 6]) // [1, 2, 3, 4, 5, 6]
                    const result = [ ...arr, 3, 4, ...[5, 6]] // [1, 2, 3, 4, 5, 6]

forEach

Метод forEach проходится по каждому элементу массива, имея доступ к элементу массива, его индексу и массиву. Всегда возвращает undefined 

                    arr.forEach( (item, index, arr) => {
                        console.log(`${index + 1}) ${item} from ${arr}`);
                    })

Методы indexOf, lastIndexOf, includes работают так же, как в строках


find, findIndex, findLastIndex

Методы возвращают значение которое выдало результат true

                    arr.find((item) => item.id === currentId)
Аналогично работают и методы findIndex и findLastIndex

filter

Метод возвращает массив из элементов вернувших true

                    arr.filter((item) => item.id === currentId)

map

Метод map вызывает функцию для каждого элемента массива и возвращает массив результатов выполнения этой функции 

                    const arr = [11, 19, 31, 19, 23]
                    const result = arr.map((item) => item * 3) 
                    console.log(arr);    // [11, 19, 31, 19, 23]
                    console.log(result); // [33, 57, 93, 57, 69]

sort, reverse

Методы sort() и reverse() сортируют и разворачивает массив 

                    arr.sort()
                    arr.reverse()

split, join

Превращают строку в массив, удаляя промежуточные символы и наоборот вставляя симполы между элементами массива 

                    const fruits = `apple, banana, orange`
                    const arr = fruits.split(`, `) // ['apple', 'banana', 'orange']
                    const str = arr.join(`./.`)    // apple./.banana./.orange

reduce

Возвращает acc который изначально равен initValue, а затем может взаимодействовать с каждым элементом массива 

                    arr.reduce( (acc, item, index, array) => {...}, initValue)

                    const arr = [1, 2, 3, 4, 5, 6]
                    const summ = arr.reduce( (acc, item) => acc + item) // 21

isArray

Для определения того, что перед нами массив typeof ничем не поможет, тк массив это частный случай объекта. Поэтому полезно использовать конструкцию Array.isArray 

                    typeof []          // object
                    Array.isArray([])  // true

Destructuring

Деструктуризация позволяет удобно объявить переменные, присвоив им значения из массива или объекта 

                const arr = [`Anna`, `Bill`, `Clint`, `Denis`, `Elvis`]
                const [user1, user2, ...users] = arr 
                // user1 = arr[0], user2 = arr[1], users = arr.slice(2)                
            
                const obj = {login: `solo`, pass: 777, age: 28, city: `Moscow`}
                const {login, pass, ...others} = obj
                // login = `solo`, pass = 777, others = {age: 28, city: `Moscow`}
